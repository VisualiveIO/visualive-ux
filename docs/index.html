toSvg

toSvg(node: Node, options: Object): Promise
Parameters
node (Node) The DOM Node object to render
options (Object) Rendering options
Name Description
options.filter Function Should return true if passed node should be included in the output (excluding node means excluding it's children as well). Not called on the root node.
options.bgcolor String color for the background, any valid CSS color value.
options.width Number width to be applied to node before rendering.
options.height Number height to be applied to node before rendering.
options.style Object an object whose properties to be copied to node's style before rendering.
options.quality Number a Number between 0 and 1 indicating image quality (applicable to JPEG only), defaults to 1.0.
options.imagePlaceholder String dataURL to use as a placeholder for failed images, default behaviour is to fail fast on images we can't fetch
options.cacheBust Boolean set to true to cache bust by appending the time to the request url
Returns
Promise: A promise that is fulfilled with a SVG image data URL
